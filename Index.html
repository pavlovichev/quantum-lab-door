Аналоговые RC-цепи

Фильтрация и обработка аналоговых сигналов в реальном времени. Используется как основа для квантовой адаптации.

// Пример кода обработки
from scipy import signal
import numpy as np

def rc_filter(input_signal, R=1, C=1):
    b = [1]
    a = [R*C, 1]
    return signal.lfilter(b, a, input_signal)
